The t-0 Network monitors transfers between the unique wallet associated with each member. It currently monitors USDT transfers on the following chains:

- Ethereum
- Binance Smart Chain
- Polygon

For more information about settlement between network members, [see Settlement with Counterparties](provider-settlement)
